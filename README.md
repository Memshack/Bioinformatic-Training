# Bioinformatic-Training
Bioinformatics Whole-Genome Alignment Training – January Workshop

This repository contains the training workflow used during a January Bioinformatics Seminar attended by over 30 undergraduate and postgraduate participants. The training covered a complete whole-genome read alignment pipeline using Escherichia coli sequencing data, starting from downloading raw reads from NCBI SRA, performing quality control, aligning reads to reference genomes using BWA, processing alignment files with SAMtools, and visualizing results in IGV. The repository serves as a beginner-friendly training pipeline for whole-genome sequencing data analysis in Linux.
